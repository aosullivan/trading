## Trust Boundaries

- The diagnostics must use the frozen focus-basket fixtures and the same `/api/chart`-compatible scoring logic, or the conclusions will not be comparable to the ratchet baseline.
- The Markdown report is a decision input for Phase 4, so hand-written conclusions without matching JSON evidence are not trustworthy enough.
