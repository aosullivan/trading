Trend Ribbon

Trend-Driven

Maintained

The main promoted ribbon-regime strategy. It uses daily ribbon flips, weekly confirmation, and cooldown logic to stay with major trends while avoiding fast re-entry after bearish breaks.

Corpus Trend

Corpus Trend

Maintained

Donchian breakout entries with ATR-aware trend discipline. Good for comparing a more classic trend-following baseline against the ribbon family.

Corpus Layered

Corpus Trend Layered

Maintained Comparison

A staged corpus variant used mainly for comparison. It behaves like corpus trend but splits exposure into layered sleeves instead of using one all-in path.

CCI Hysteresis

CCI Hysteresis

Maintained

The current promoted focus-basket baseline. It enters only after strong CCI expansion and exits with a looser lower threshold, which makes the long bias stickier than the basic CCI trend model.

Polymarket

Polymarket Skew

Maintained

Prediction-market driven direction using saved Polymarket probability history. This is most relevant on `BTC-USD`, where the external signal is available.

Weekly Core

Weekly Core + Daily Overlay

Experimental

Keeps a slower weekly core allocation on while the higher timeframe stays bullish, then uses a faster daily overlay for the last sleeve of exposure.

Supertrend-I

Supertrend-I

Experimental

A touch-triggered Supertrend variant. It uses the same ATR ratchet family, but flips when the candle touches the active band instead of waiting for a close beyond it.

EMA 9/26

EMA 9/26 Cross

Experimental

A slower EMA crossover variant tuned for smoother index-style trends. It keeps the comparison cleaner than the faster 5/20 pair and supports the same daily-plus-weekly confirmation flow.

Semis Persist

Semis Persist v1

Experimental

A semiconductors-tuned persistence model: a slower 30/100 EMA trend stack gets the position on, then exits only after 10 straight bearish bars so leader names have more room to keep running.

BB Breakout

BB Breakout

Experimental

Volatility breakout entry using Bollinger Bands with a simpler exit rule. Useful when you want to compare the maintained trend models to a cleaner breakout family.

EMA 5/20

EMA 5/20 Cross

Experimental

The original faster EMA crossover momentum model. It is still on the chart as the quicker comparison line, while `EMA 9/26` now covers the slower medium-trend version.

CCI Trend

CCI Trend

Experimental

The simpler CCI breakout version without hysteresis. It is kept around as the clean reference line for the stronger `CCI Hysteresis` variant.
